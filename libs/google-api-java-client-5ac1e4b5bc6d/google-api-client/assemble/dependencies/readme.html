Dependent libraries for the ${project.name} ${project.version}

When building an application based on this client library, use these dependent library jars:

  • jackson-core-asl-1.5.3.jar: needed for JSON, or formats based on JSON like JSON-C, JSON-RPC, or POCO. It can be downloaded from here.

When building an application for all Java environments except for Android (whose SDK already includes these), additionally use these dependent library jars:

  • kxml2-2.3.0.jar: needed for XML, or formats based on XML like Atom. It can be downloaded from here.
  • commons-logging-1.1.1.jar, httpcore-4.0.1.jar, and httpclient-4.0.1.jar: needed only if you prefer to use the Apache HTTP library for the underlying HTTP transport (default is to use the built-in java.net package of the JDK). They can be downloaded from here. Warning: Do not use the Apache HTTP library on App Engine because it won't work. The Apache HTTP library tries to open its own sockets which is not allowed on App Engine.
